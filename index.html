Frequently Asked Questions

Find quick answers to common questions

What is Trezór Bridge?

Trezór Bridge is a communication tool that enables secure connection between your Trezór device and web browsers. It handles device detection, firmware updates, and encrypted data transfer.

Is it necessary to install Trezór Bridge?

Yes, Trezór Bridge is required for using Trezór Suite and Trezór Wallet in your web browser. It ensures secure communication between your device and these applications.

How do updates work?

Trezór Bridge automatically checks for updates and installs them in the background. You'll always have the latest security features without manual intervention.

Is Trezór Bridge secure?

Yes, Trezór Bridge is digitally signed by SatoshiLabs and uses end-to-end encryption for all communications. Your private keys never leave your Trezór device.
